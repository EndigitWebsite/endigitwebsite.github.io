Support

Endigit's goal is to help our customers be successful.  If you need support for work done by Endigit or for IP Blaster, then please contact us.  The Endigit Support Tool below may be useful to share you computer screen with an Endigit engineer.

Endigit Support Tool

The Endigit Support Tool may be run on any computer connected to the internet.  It provides a way of allowing an Endigit engineer to view and control your computer remotely.  It only allows viewing and controlling once the Engineer has a 6 digit code.

Instructions:

  1. Download Endigit Support.ex_
  2. Replace ".ex_" extension with ".exe"
  3. Run "Endigit Support.exe"
  4. Contact an Endigit engineer with the six digit number

Contact support@endigit.com for support issues.
